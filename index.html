🟢 Nivel 1: Introducción a las Bases de Datos

Comprender qué es una base de datos, su utilidad y tipos principales

¿Qué es una base de datos?

Una base de datos es un conjunto organizado de información que puede ser fácilmente accedida, gestionada y actualizada. Es como una biblioteca digital donde cada libro tiene su lugar específico y puede ser encontrado rápidamente.

¿Para qué sirve?

Las bases de datos nos permiten almacenar y recuperar información eficientemente. Por ejemplo:

  • Usuarios de una aplicación móvil
  • Notas escolares de estudiantes
  • Inventario de una tienda
  • Información médica de pacientes

Tipos comunes de bases de datos

  • Bases de datos relacionales: MySQL, PostgreSQL, Oracle
  • Bases de datos no relacionales: MongoDB, Redis
  • Locales vs. en la nube: Almacenadas en tu computadora o en internet

Componentes esenciales

  • Dato: Unidad mínima de información (ej: 23, "Juan")
  • Registro: Conjunto de datos relacionados (ej: Juan, 23, Estudiante)
  • Tabla: Colección de registros similares
